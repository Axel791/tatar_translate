# Стайлгайд: перевод «Достижений» лауреатов на татарский язык

Жанр исходников: наградные списки с датами, служебные характеристики, анкетные фрагменты.
Цель: **смысловой, человекочитаемый литературный татарский**, официально-нейтральный тон.
Не дословно, а по смыслу. Ничего не выдумывать и не терять по содержанию.

## Грамматика и стиль
- Изафет II типа для именных цепочек: определяемое получает аффикс принадлежности
  («Мактау грамотасы», «авыл хуҗалыгы хезмәткәре», «Мамадыш муниципаль районы»).
- Естественный татарский порядок слов: сказуемое в конце («...медале белән бүләкләнде»).
- Повествование о фактах — прошедшее на -ды/-де («танытты», «катнашты»);
  перечисление наград без повествования — на -ган («бүләкләнгән», «бирелгән»).
- Кавычки «ёлочки»; вложенные — “лапки”: «Красногорский “Алёнушка” балалар бакчасы».
- Первая буква — заглавная. ё сохраняется (контролёр, режиссёр).
- Тире — длинное с пробелами ( — ).

## Аббревиатуры
- РТ / Республика Татарстан → Татарстан Республикасы (при повторных упоминаниях можно ТР)
- РФ / Российская Федерация → Россия Федерациясе (повторно — РФ)
- СВО → махсус хәрби операция; ВОВ / Великая Отечественная война → Бөек Ватан сугышы
- АО → АҖ; ОАО → ачык акционерлык җәмгыяте; ПАО → ГАҖ; ООО → ҖЧҖ
  (порядок: «Татэнерго» АҖ, «ТНГ-Групп» ҖЧҖ — имя в кавычках первым)
- КФХ → крестьян (фермер) хуҗалыгы · ФАП → фельдшер-акушерлык пункты
- ДШИ → балалар сәнгать мәктәбе · ДЮСШ → балалар-яшүсмерләр спорт мәктәбе
- ЦРБ → үзәк район хастаханәсе · СОШ (в полном названии) → урта гомуми белем бирү мәктәбе
- Служебные аббревиатуры организаций (МКУ, МБУ, МБОУ, МБДОУ, ГАУЗ, ФГБОУ ВО, ПАО «КАМАЗ» и т.п.)
  остаются как в исходнике, название переводится: «Яшел Үзән музыка театры» МБУ.
- Министерства: Министерство сельского хозяйства РФ → Россия Федерациясе Авыл хуҗалыгы министрлыгы;
  Министерство образования и науки РТ (МОиН РТ) → Татарстан Республикасы Мәгариф һәм фән министрлыгы;
  Министерство здравоохранения → Сәламәтлек саклау министрлыгы.

## Числа, даты, имена
- Даты в цифровом виде не менять: 24.03.2021, 02.2019, «с 28.10.2021 по 20.09.2022» → «28.10.2021 — 20.09.2022 чорында».
- Месяцы словами: январь → гыйнвар (остальные месяцы пишутся как в русском):
  «с 11 января 2009 года» → «2009 елның 11 гыйнварыннан».
- «с 2005 года» → «2005 елдан», «в 2025 г.» → «2025 елда», «2020-2026 г.г.» → «2020–2026 еллар».
- Порядковые с цифрой: «5 нче разрядлы», «1 нче категорияле» (1,2,3,4,5,7,8 → нче; 6,9,10,40,60,90 → нчы).
- «80 тыс.» → «80 мең»; проценты как в исходнике (81%).
- **Имена и отчества татарского (тюрко-мусульманского) происхождения писать в татарской орфографии**:
  Альфия → Әлфия, Кабировна → Кәбировна, Гузель → Гүзәл, Гузелия → Гүзәлия, Рамиль → Рамил,
  Ильдар → Илдар, Ильназ → Илназ, Ильсур → Илсур, Ильшат → Илшат, Наиль → Наил, Фарит → Фәрит,
  Фанис → Фәнис, Рустем → Рөстәм, Гульнара → Гөлнара, Гульназ → Гөлназ, Айгуль → Айгөл,
  Лейсан → Ләйсән, Зульфия → Зөлфия, Шамиль → Шамил, Равиль → Равил, Миляуша → Миләүшә,
  Махмутович → Мәхмүтович, Миннахметович → Миңнәхмәтович, Тауфикович → Тәүфыйкович, Хакимович → Хәкимович.
  Если точная татарская форма имени НЕ очевидна — оставить написание исходника (не изобретать).
- Русские/славянские имена (Максим Александрович, Марина Павловна, Виктор Николаевич) — без изменений.
- Инициалы не менять (Р.И. остаётся Р.И.). Фамилии оставлять в написании исходника (паспортная форма):
  «Сулейманова Р.И.» → «Сулейманова Р.И.».
  В связном тексте имя в именительном падеже + татарская конструкция («Рамил Мәхмүтович җитәкчелегендә»).

## Названия конкурсов, премий, изданий
Переводить только при общеизвестном татарском варианте («Алишевские чтения» → «Алиш укулары»,
«Звёзды будущего» → «Киләчәк йолдызлары»). Если официальный татарский вариант не известен точно —
оставить название как в исходнике, в кавычках «». Названия брендов и техники не переводить.

## Краевые случаи
- **Обрезанный исходник** (обрыв на полуслове, часто ~500 знаков): перевести всё осмысленное;
  финальное оборванное полуслово/огрызок без восстановимого смысла — опустить. НЕ досочинять.
- **Анкетные фрагменты**: «1.Какими государственными наградами награжден(а), дата награждения:» →
  «1. Нинди дәүләт бүләкләре белән бүләкләнгән, бүләкләү датасы:» — структуру и нумерацию сохранять.
- **Опечатки исходника** (комлпекса, Имет, пропущенные пробелы) — не воспроизводить, переводить по смыслу.
- **Мусор без смысла** («О», «-», «.») — скопировать как есть.
- Слово-заглушка «Достижения» → «Казанышлар».
- Двойные/висячие запятые исходника («, ,») — привести к нормальной пунктуации.

## Эталонные примеры (следовать этому стилю)

**Проза-характеристика:**
- RU: «Зарекомендовал себя как надежный, ответственный работник, не нарушающий сроки выполнения заданий и предоставляющий исчерпывающую отчетность. В трудных ситуациях быстро принимает оптимальное решение. В коллективе пользуется уважением и хорошим отношением коллег.»
- TT: «Үзен ышанычлы, җаваплы хезмәткәр итеп танытты: биремнәрне үтәү срокларын бозмый, тулы хисап бирә. Авыр хәлләрдә тиз арада оптималь карар кабул итә. Коллективта хезмәттәшләренең хөрмәтен һәм яхшы мөнәсәбәтен казанган.»

**Наградной список с датами (пунктуацию упорядочить: награда, дата; …):**
- RU: «звание «Заслуженный работник сельского хозяйства Республики Татарстан», 31.05.2004, , Благодарственное письмо Министерства сельского хозяйства Российской Федерации, 02.07.2014, медаль «За доблестный труд», 16.02.2017»
- TT: ««Татарстан Республикасының атказанган авыл хуҗалыгы хезмәткәре» исеме, 31.05.2004; Россия Федерациясе Авыл хуҗалыгы министрлыгының Рәхмәт хаты, 02.07.2014; «Фидакарь хезмәт өчен» медале, 16.02.2017»

**Анкетный фрагмент:**
- RU: «1.Какими государственными наградами награжден(а), дата награждения: … 2. Другие виды поощрений, награждений: …»

**Организация в названии награды:**

- TT: ««Транснефть – Прикамье» акционерлык җәмгыятенең Әлмәт район нефть үткәргечләре идарәсе Мактау тактасы»

**СВО:**
- RU: «Принимал участие в специальной военной операции, за период выполнения боевых задач зарекомендовал себя как отважный и умелый боец»
- TT: «Махсус хәрби операциядә катнашты, сугышчан бурычларны үтәгән чорда үзен кыю һәм оста сугышчы итеп танытты»

**Татаризация имени + топонимы + месяц (обязательный образец):**

Ул үзенең һөнәри дәрәҗәсен һәрвакыт күтәрә, яңа эш алымнарын үзләштерергә омтыла. Яшел Үзән муниципаль районының 1576 нчы сайлау участогы рәисе булып тора, 2009 елдан сайлау кампаниясендә катнаша.»

## Формат ответа (для батч-перевода)
На каждый входной текст — ровно одна строка JSON:
`{"k": "<ключ>", "tt": "<перевод>"}`
Без комментариев, без пропусков, без изменения ключей. Перевод — одна строка (без \n).
